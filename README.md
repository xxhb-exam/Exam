# Exam 考试系统
在大三上期完成的期末实训。一个简单在在线考试系统。分为管理出题和考试界面考试。
***
# 鸣谢
***
我们团队开发所有人员。
# 内置功能
***
- 后台
1. 试卷管理（增、删、改、查）
2. 题库管理（增、删、改、查）
3. 成绩查看(查)
4. 用户查看(查)
- 前台
1. 选择试题
2. 考试界面（考试完出成绩,考试完查看答案）。
3. 查看成绩
#技术选择
***
- 后端
1. Spring
2. Spring MVC
3. MyBatis
4. 阿里云 Druid 数据库连接池
5. FreeMarker 模板引擎
6. pagehelper分页插件
- 前端
1. Bootstrap
2. jQuery
3. Vue.js
4. Datatables
***
